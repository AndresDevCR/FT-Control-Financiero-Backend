  # DMS Backend

## Description

This is the backend for the DMS project. It is built using NestJS, making use of the TypeORM library for database management.

## Installation

1. Up docker-compose.yml

```
docker-compose up -d
```

2. Install dependencies

```
npm install
```

3. Run

```
npm run start:dev
```

## Test

```
npm run test
```

## Production

```
npm run start:prod
```
